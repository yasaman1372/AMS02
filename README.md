## Analysis Overview

This repository contains two main analysis pipelines developed for AMS-02 data analysis:

## 1. Electron Analysis

The electron_analysis pipeline focuses on the measurement and characterization of high-energy electron fluxes using AMS-02 data.

The workflow includes:

* Processing and selecting electron events from AMS-02 data
* Applying event selection criteria and optimization of analysis cuts
* Performing template fits to separate signal and background contributions
* Calculating electron fluxes
* Performing statistical analysis and evaluating analysis results

## 2. Photon Analysis

The photon_analysis pipeline focuses on the analysis of high-energy photons from AMS-02 data.

The workflow includes:

* Processing and selecting photon candidate events
* Applying selection criteria for photon event identification
* Estimating backgrounds and extracting photon signals
* Calculating photon fluxes and related quantities
* Performing statistical analyses
* Generating sky maps to study the spatial distribution of photon events
* Investigating astrophysical gamma-ray sources and searching for potential signals or transient activity

Together, these pipelines demonstrate experience in scientific data analysis, event reconstruction, statistical methods, and the development of automated analysis workflows for large experimental datasets.
